Jak korzystać z repozytorium

Proces i dokumentacja

Plan faz, zadania oraz notatki znajdziesz w katalogu docs/. Aktualne decyzje architektoniczne trzymamy w docs/adr/.

Najważniejsze pliki: docs/plan.md, docs/tasks.md, docs/notes.md.

Automatyzacja

Repo zawiera testy w tests/ oraz workflow GitHub Actions (.github/workflows/codex.yml) uruchamiający pytest.

Cel: utrzymać spójność nawigacji i jakości strony na GitHub Pages.

Repozytorium jest gotowe do hostingu na GitHub Pages — wystarczy skierować publikację na gałąź z plikami statycznymi.

Próby Płomienia

Trzy progi inicjacyjne prowadzą od pierwszej iskry po finałowe płomienie zakonu. Każda karta łączy fotografie archiwalne z repozytorium i kieruje do pełnych opracowań.

Dodaj pomysł

Masz iskierkę fabularną, mechanikę rytuału albo scenę do dopisania? Zostaw ją tutaj — zwiadowcy archiwum zapiszą ją do dalszej kuracji.

Pomysły trafiają do archiwum tylko dla zespołu mistrzów gry. Oddziel słowa kluczowe przecinkami.
